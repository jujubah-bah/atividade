# Exercício
Atividade em grupo!!
Grupo: Julia, Larissa, Andriel

Algorítimo para comprar ingressos no cinema para filmes com classificação indicativa.

De início, o algorítimo pedia o preço do ingresso, a quantidade de pessoas e calculava o preço a ser pago. Porém, mudamos para um algorítimo com um preço fixo no ingresso, perguntas individuais e a escolha de filmes que estão em cartaz nos cinemas atualmente. O diferencial deste algorítimo é o armazenamento da informação sobre a idade do cliente, que, se tiver uma idade menor que a classificação indicativa dos filmes, não poderá assistir estes.
